## Folder structure 
The project's folder structure follows the #1 folder structure mentioned [here](https://unity.com/how-to/organizing-your-project)

- Art: Contains all the 
  - [Materials](https://docs.unity3d.com/Manual/Materials.html)
  - [Models](https://docs.unity3d.com/560/Documentation/Manual/FBXImporter-Model.html)
  - [Textures](https://docs.unity3d.com/Manual/Textures.html)
- Audio: Contains all the audio related to the game
  - Music: Game music, usually longer
  - Sound: sound effects, usually shorter
- Code:
  - Scripts: C# scripts
  - Shaders: Customs shader files
- Docs: All files related to documentation
- Level: All the files related to game desing
  - [Prefabs](https://docs.unity3d.com/530/Manual/Prefabs.html)
  - [Scenes](https://docs.unity3d.com/Manual/CreatingScenes.html)
  - UI: All files related to UI
